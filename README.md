# Piping Business Agent — Comprehensive WEB DASHBOARD

A powerful, easy to build business agent for piping/sanitaryware distributors. Everything you need in one place: proposals, invoicing, delivery challans, reminders, & stock tracking.

## Key Features

- **Sell** - Generate professional two-sided PPTX proposals in seconds
- **Invoice** - Create GST compliant sales voucher XML to post to Tally directly
- **Deliver** - Generate B form DC PDF &n automatically deduct stock
- **Reminder** - Track due payments, low stock, proposal follow-ups
- **Dashboard** - Real-time stats, inventory, client filters
- **Zero SEtup** - Pre-filled with dummy data. Zust enable your MBD 📷
